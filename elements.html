Projects Page

Search Engines

Engineered a search engine in C++ that processes text data to build an inverted index, enabling efficient query handling. Implemented text cleaning and tokenization to remove punctuation and normalize words.

The engine supports complex search queries using set operations like union, intersection, and difference, to return relevant search results. Developed the ability to handle large datasets with optimized data structures such as maps and sets, showcasing proficiency in file I/O, string manipulation, and algorithmic design for search functionality.

UIC CS Student App

Developed a python based program that serves the specific needs of UIC Computer Science students, aiding in their academic journey and fostering a sense of community.

Included information such as UIC CS degree audits and plans, ways to get around campus and discover new places, as well as CS Based clubs for students to discover.

All About Allstate

Ciphers

The “Ciphers” project is an educational initiative aimed at exploring various encryption techniques and their applications in cybersecurity. This project involves understanding, implementing, and analyzing different cipher algorithms, highlighting their strengths, weaknesses, and use cases.

Utilized the implementation of Caeser Cipher and Vigenere Cipher and allowed for user to decrypt and encrypt words and sentences of their choice.

Food Web Analysis

Developed a comprehensive C program to model ecological food webs, focusing on dynamic memory management. Implemented features to build and modify the web structure, including predator-prey relationships, using dynamic arrays and pointers.

The program supports analysis of ecological roles (such as apex predators and producers), dietary flexibility, and impacts of species extinction on the web. Integrated command-line arguments for customizable program execution modes and ensured efficient memory usage with meticulous allocation and deallocation practices.
